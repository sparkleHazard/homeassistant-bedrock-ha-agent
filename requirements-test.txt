# HA component pypi-deps that our integration drags in at import time.

# HA core's pypi package ships without per-component requirements; those live
# in each component's manifest.json and only get installed when a real HA

# `custom_components.bedrock_ha_agent.__init__` which chains
# `bedrock_client` → `vision` → `homeassistant.components.{camera,stream,conversation}`,
# each of which imports its manifest deps at module load time. We have to
# declare those deps here or collection fails with ModuleNotFoundError.
#
# conversation (manifest): hassil, home-assistant-intents

# stream     (manifest): PyTurboJPEG, av (bundles ffmpeg inside its wheel)
